<h1>Weather-App</h1>
<p>A clean and interactive weather application that provides real-time weather updates for any city. Built using HTML, CSS, and JavaScript, this app features dynamic UI updates, temperature unit conversion, and demonstrates API integration.</p>

<p align="center">
  <img src="screenshots/UI.png" width="600"/>
</p>

## Features
- City-based weather search  
- °C / °F temperature toggle  
- Dynamic weather-based backgrounds  
- Custom weather icons  
- Enter key search support  
- Saves last searched city  
- Real-time API data  
- Error handling
